疑義注記

疑義與版本注記

① 產品命名會演變

BBrowser → BBrowserX；Lens → SpatialX。本教程以 2026 年官網命名為準。若你看到舊名（BBrowser、Lens），指的是同系列產品的早期版本。教程中的 GUI「點選路徑」為概念示意，實際選單名稱可能因版本而異——請以你安裝版本的官方文件為準。

BBrowser → BBrowserX; Lens → SpatialX. This tutorial follows the 2026 site naming. Older names (BBrowser, Lens) refer to earlier versions of the same line. GUI "click paths" here are conceptual — exact menu labels may vary by version; defer to the official docs for your installed build.

② 官方數字會變動

如「1 億+ 參考細胞」「54 類 / 183 亞型」「Talk2Data 120M+ 細胞」等，皆為官網於特定時點公佈的數字，會隨資料庫更新而變。引用於論文時請標註取用日期並回查官網最新值。

Figures like "100M+ reference cells", "54 types / 183 subtypes", and "Talk2Data 120M+ cells" are point-in-time marketing numbers that grow with the database. Cite with an access date and re-check the site for current values.

③ 「Signac」名稱衝突

BioTuring 內建 Venice 的分析套件名為 Signac，與 Stuart 實驗室廣為使用的單細胞 ATAC-seq 套件 Signac（Seurat 生態系）並非同一個東西。查資料時務必辨明語境。

BioTuring's analytics package that houses Venice is named Signac — not the same as the widely used scATAC-seq package Signac from the Stuart lab (Seurat ecosystem). Disambiguate by context when searching.

④ Monocle 版本

BBrowserX 的軌跡分析文獻多引用 Monocle 2（reversed graph embedding），與後續改版的 Monocle 3 演算法不同；若比較結果差異，先確認版本。

BBrowserX's trajectory analysis is generally cited as Monocle 2 (reversed graph embedding), which differs from the later Monocle 3; if results differ, check the version first.

⑤ GUI 便利 ≠ 免於判斷

自動註釋、預設閾值與一鍵分析降低了操作門檻，但生物學判讀責任仍在使用者。務必用 marker、生物學情境與（可行時）程式碼工具交叉驗證關鍵結論。

Auto-annotation, default thresholds, and one-click analysis lower the barrier, but biological interpretation remains the user's responsibility. Cross-validate key conclusions with markers, biological context, and (where feasible) code tools.
